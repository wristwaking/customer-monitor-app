# 微信客服数据监控桌面控制面板
这是一款基于electron框架搭建微信客服数据监控桌面控制面板。通过记录微信群客服会话数据，支持脚本快速指定日期统计导出数据。

**关联项目：https://github.com/wristwaking/customer-monitor-wechat-bot**
